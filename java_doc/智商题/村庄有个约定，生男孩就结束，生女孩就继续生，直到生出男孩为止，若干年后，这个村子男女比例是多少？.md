# ✅村庄有个约定，生男孩就结束，生女孩就继续生，直到生出男孩为止，若干年后，这个村子男女比例是多少？



# 典型回答  

"直到生出男孩为止"说明了：

 (1) 每家生了男孩后便不再生了 

 (2) 每家都只有一个男孩 



 那么我们可以先求出每家孩子个数的期望设为X 

 X       P(概率) 

 1        0.5 //第一次就生出了男孩的概率是0.5 

 2        0.5^2 

 ... 

 E(X) = 1*0.5+ 2*0.5^2+...= 1/0.5=2



每家孩子个数的期望是2，而且都有且只有1个男孩，那么女孩的个数也是1



所以男孩和女孩个数相等！
